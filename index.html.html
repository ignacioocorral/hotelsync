Configuración de base de datos
①
Crea una cuenta gratis en supabase.com, crea un proyecto nuevo y ejecuta el SQL de abajo en el SQL Editor.
SQL — pega esto en Supabase → SQL Editor → New Query
create table if not exists hotelsync_store (
  key text primary key,
  value jsonb not null,
  updated_at timestamptz default now()
);
alter table hotelsync_store enable row level security;
create policy "public_access" on hotelsync_store
  for all using (true) with check (true);
Conectar tu proyecto
Los encuentras en: Supabase → Project Settings → API
